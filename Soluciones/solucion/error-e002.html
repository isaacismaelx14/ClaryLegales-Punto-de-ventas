El valor incertado no es valido

Este error ocure porque el programa de ClaryLegales punto de ventas no ha podido reconocer el código establecido en la casilla código.

causas/solucion

  • Has dejado la casilla "código" en blanco o has introducido algun codigo invalido dentro de ella
    • 1. Introduce un codigo valido dentro de la casilla. ejm: 1

    Como puedes ver en el ejemplo hemos utilizado el código 1

    Recomendaciones

    Códigos validos

    Un código valido, son la serie de numberos utilizados para identificar un producto. Cada producto contine un código único el cual lo identifica en la base de datos. Para saber cuales son los códigos validos puedes entrar a la ventana productos y mirar la lista de códigos con sus respectivos productos.

    Nota:

     Los códigos se componen solo de números, esta casilla no asepta: letras, espacios ni caracteres especiales.
